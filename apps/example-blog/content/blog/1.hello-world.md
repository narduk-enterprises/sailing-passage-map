# Hello World

Welcome to the **Nuxt 4 + Nuxt UI 4 Edge Template**. We are excited to showcase how you can build lightning-fast content sites.

## The Power of the Edge

By leveraging Cloudflare Workers, we eliminate Node.js cold starts. When combined with _Nuxt Content v3_, your markdown files are queried efficiently via D1 at the edge.

### Features

- Native Markdown parsing
- MDC (Markdown Components) support
- Built-in syntax highlighting
